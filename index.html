⚡ Vercel
Miễn phí · Nhanh nhất · Khuyến nghị

Hosting tĩnh miễn phí. Tốc độ CDN toàn cầu. Tự động HTTPS.

Bước 1. Vào vercel.com → Sign Up
Bước 2. Chọn "Continue with GitHub" (tạo GitHub nếu chưa có)
Bước 3. Click Add New Project → Upload
Bước 4. Kéo thả thư mục school-app vào
Bước 5. Click Deploy → Chờ 30 giây → Nhận link 🎉
🚀 Mở Vercel ngay
📄 GitHub Pages
Miễn phí · Dễ cập nhật

Hosting trực tiếp từ GitHub. Cập nhật = commit file mới.

Bước 1. Vào github.com → Sign Up
Bước 2. Click New repository, đặt tên school-hp, chọn Public
Bước 3. Upload file index.html lên repo
Bước 4. Settings → Pages → Source: main branch
Bước 5. Link: tenban.github.io/school-hp
📂 Mở GitHub ngay
☁️ Netlify Drop — Cách nhanh nhất (kéo thả là xong)
1. Vào app.netlify.com/drop
2. Kéo thả thư mục school-app vào vùng màu xanh
3. Chờ 10 giây → Có link ngay (VD: random-name.netlify.app)
4. Đăng ký tài khoản để giữ link cố định
5. ✓ Hoàn toàn miễn phí, không cần biết lập trình
